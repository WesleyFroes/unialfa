Introdução

Este projeto, "Primeiro Site", foi construído utilizando as linguagens HTML5, CSS e JavaScript com o objetivo de mostrar conceitos de Paradigmas de Programação, Git e GitHub. Conceitos estes pesquisados por nós e dispostos no menu acima. Fique à vontade para clicar nos links e explorar os conceitos e a estruturação do código.
Vale a pena lembrar que o outro objetivo é mostrar na prática a utilização das linguagens acima citadas bem como seus conceitos e semânticas. Tentamos deixar o máximo possível de comentários no código para aqueles que quiserem explorar com mais detalhes a construção do mesmo possam se beneficiar.
Então fique à vontade para inspecionar o código em seu navegador ou simplesmente abri-lo no Visual Studio Code ou com seu editor de preferência. Desejamos uma boa experiência!
